Q : อธิบาย Data Flow ของโครงสร้าง backend

A : ในโปรเจกต์จะแยกโครงสร้างเป็น Layer ชัดเจนครับ เริ่มจากวิ่งผ่าน Middleware เพื่อเช็กสิทธิ์แอดมินก่อน จากนั้น Routes จะส่งต่อไปที่ Controller เพื่อคุม Logic ส่วนการคุยกับฐานข้อมูล เราจะแยกเขียนไว้ในชั้น Services โดยใช้ Prisma ครับ การแยกแบบนี้ทำให้โค้ดเป็นสัดส่วน ปลอดภัย และบำรุงรักษาง่ายครับ
